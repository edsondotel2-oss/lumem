# 🌟 Lumem

**Sua luz para a vida espiritual**

Lumem é um aplicativo web completo e gratuito que reúne a Bíblia Sagrada, a Liturgia Diária, o Santo Terço e um acervo de orações em uma interface moderna, acolhedora e responsiva. Acessível de qualquer dispositivo com um navegador.

## ✨ Funcionalidades

- 📖 **Bíblia Completa**: Navegue por livros, capítulos e versículos. Busque por palavras ou temas. Marque seus versículos favoritos (salvos no seu navegador).
- 📅 **Liturgia Diária**: Acompanhe as leituras do dia, salmos e o evangelho (em breve com integração a API pública).
- 🙏 **Santo Terço**: Guia interativo com os mistérios e contagem das orações (em desenvolvimento).
- 🕊️ **Orações**: Catálogo de orações organizado por categorias (em desenvolvimento).
- ⚙️ **Configurações**: Personalize o tema (claro/escuro) e o tamanho da fonte (em breve).

## 🚀 Tecnologias

- **Python 3.13+** – Linguagem principal.
- **Flet** – Framework para criar interfaces web e aplicativos multiplataforma com Material Design 3.
- **Client Storage** – Favoritos e preferências salvos localmente no navegador (sem necessidade de cadastro).
